Decorators¶

django_roles.decorator.access_by_role(view)[source]¶

Check if logged user can access the decorated function or class.

A user (django.contrib.auth.models.User) can access the decorated function or class when it belong to a django.contrib.auth.models.Group with access given by an object of type roles.models.ViewAccess.

PermissionDenied is raised if user has no access.
